① Wordiness trap 3 pts

Rewrite each long, wordy sentence into a concise one using exactly one of today's chunks.
T1.After a long working day, I usually spend some time doing relaxing things so that my body and mind feel less stressed.
T2.Because I was really tired, I accidentally started to fall asleep for a short time while I was watching TV.
T3.I have wanted to clean my room for a long time but I kept delaying it and only finally did it yesterday.
